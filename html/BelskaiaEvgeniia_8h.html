Котик. Треугольный котик. Поскольку кот живет на другой планете, усов у него пока нет.
Зато уши, лапы и хвост (нос и глаза + зрачки) присутствуют и могут выражать настроение котика

Аргументы
x,y- координаты середины шеи кота.
UpHvost- поднимает хвост кота
GoNose- увеличивает нос во все стороны
sizeLeftEye- увеличивает левый глаз
sizeRightEye- увеличивает правый глаз
UhoLDown- двигает левое ухо
UhoRDown- двигает правое ухо
ZrachkiL- перемещает левый зрачок влево/право (возможно сделать сумашедшего кота, запустив зрачек по кругу...)
ZrachkiR- перемещает правый зрачок влево/право
LlapkaUp- поднимает левую лапку
RlapkaUp- поднимает правую лапку
Yazik- показывает язык
colorCat- цвет всего кота и обводки
colorNose- цвет носа
colorLapki- цвет лапок (желательно, чтобы цвет отличался от основного цвета кота)
colorEyes- цвет глаз
Пример:
DrawCat (600, 400, 7, 1, 2, 2, 0, 0, 0, 0, 0, 0, 0, TX_GREY, TX_RED, TX_BLACK, TX_GREEN);
